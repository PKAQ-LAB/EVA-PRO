## 运行前需要安装

- git
- nodejs
- yarn

## 使用方式

```bash
$ git cloneaddInterceptors
$ yarn install
$ yarn start         # visit http://localhost:8000
```

## 设置国内源 or 镜像

**设置淘宝镜像** yarn config set registry https://registry.npm.taobao.org --global

**国内源** \$ npm i yarn tyarn -g
